Key Features

  1. Create high quality spatial memory tasks without prior programming knowledge.
  2. Create customized trial types by manipulating key environmental features including arena size, colour, shape and goals.
  3. Create custom trial blocks that allow for both trial randomization and learning to criterion.
  4. Place objects/goals throughout the environment discretely, or by providing a chosen distribution area.
  5. Easily use your own 3D object sets.
  6. Capture accurate behavioural data up to 60 times per second
  7. Easy to analyze data sets with output to .csv files
